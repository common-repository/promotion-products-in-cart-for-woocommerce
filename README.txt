== Description ==

This plugin will give the option to the store owner to promote there products on the Cart Page.

It will easy for customer to pick any product directly from the cart page instead to shop page or else. Store owner have many option at backend to how to show products on cart.

Features
---------------------------------------
1) Show your product directly into cart page to boost your sell.
2) Show product by Best selling, By sale product, Featured Products, Newest Products.
3) Show your product at cart page by Specific category too.
4) Set Product limit and product grid column.
5) Set product order by Name, ID, Date.
6) Set custom Label for promotion section

== Installation ==
From WordPress Dashboard go to “Plugins >> Add New >> Uploads”, select ‘promotion-product-cart.zip’ file and upload it. Or,

Extract the zip file, copy and paste to the ‘wp-conten/plugins/’ directory of your WordPress Installation.

Activate the plugin.
